## Installation
Make sure you have Python 3.10+ installed.

```bash
git clone https://github.com/jermeinthomas/JermeinBot-v2
cd JermeinBot-v2
pip install -r requirements.txt

How to run 
bash
python bot.py

Why This Project Matters

This project is built with simplicity in mind, but with enough structure to demonstrate good practices:

separation of concerns

clean function design

readable code layout

maintainable file organization

It serves as a solid foundation for beginners improving their Python skills and for junior developers building their portfolio with practical, well-organized code.
